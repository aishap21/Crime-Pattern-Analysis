# Crime Pattern Analysis and Hotspot Mapping

## 📌 Project Overview
This project analyzes crime data to identify patterns and visualize crime hotspots using data science and machine learning techniques.

## 🚀 Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Crime clustering using K-Means
- Heatmap visualization of crime hotspots
- API integration for accessing results

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Folium (for maps)
- Flask (API)

## 📂 Project Structure
- src/ → Core implementation
- api/ → Backend API
- maps/ → Generated heatmaps
- data/ → Dataset (not included due to large size)

## ⚠️ Note
Dataset is large and not uploaded to GitHub. It can be provided separately if required.

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python src/app.py

## 📊 Output
- Crime hotspot heatmap
- Clustered crime regions
